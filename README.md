About

System Requirements
-------------------
This plugin is compatible with:
 - OMP 1.2.0

Installation
------------
To install the plugin:
 - Upload the tar.gz file in OMP (Management > Website Settings > Plugins > Generic Plugins)
 - Clone the code from Github in the plugin/generic folder.
